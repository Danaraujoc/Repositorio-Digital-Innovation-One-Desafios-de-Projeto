O que é Estrutura de Dados?
É uma estruturas organizada de dados na memoria de um computador, de forma que os dados possam ser utilizados de forma correta.

Algoritmo
É um conjunto de instruções estruturadas e ordenadas, com objetivo de realizar uma tarefa ou operação específica. São utilizadas para manipular dados.

Principais Estruturas de Dados:

Vetores e Matrizes [Arrays]
Registro
Lista
Pilha
Fila
Árvore
Tabela Hash
Grafos

Vetores e Matrizes
Vetores:
São estruturas de dados simples que podem auxiliar quando há muitas variáveis do mesmo tipo em um algoritmo.

Matrizes ou array  multi-dimensional é um vetor de vetores, ou vetor que possui duas ou mais dimensões

Registro é uma estrutura de dados especial que permite armazenar mais de um tipo de dado 

Estrutura de Dados (Listas)
Armazena dados de um determinado tipo em uma ordem específica.
Diferença entre listas e arrays e que as listas possuem um tamanho ajustável, enquanto arrays possuem tamanho fixo.

Dois tipos de listas 

Ligadas com amarração ao próximo nó

Duplamente ligada é bidirecional com amarração com o próximo nó e nó anterior

Pilha é uma estrutura que permite acesso ao item some um por vez.

Dois tipos de pilha:
LIFO (last in first out) Ultimo que entra primeiro que sai.
FIFO (first in first out) Primeiro que entra primeiro que sai.


Filas na estrutura do tipo fila o primeiro elemento a ser inserido é o primeiro elemento a ser retirado.

Estrutura de Dados (Árvore)

É uma estutura de dados  que organiza seus elementos de forma hierárquica, onde existe um elemento que fica no topo chamado de "raiz", e os elementos subordinados chamados de "nós ou folhas"

Estrutura de Dados (Hash) 

É uma estrutura de dispersão ou espalhamento, é uma estrutura que associa chaves de pesquisa a valores.
É uma generalização da idéia de array, que utiliza uma função chamada "Hashing" para espalhar os elementos fazendo com que os mesmos fiquem de forma não ordenada dentro do "array" que define a tabela.

Ela permite  associação de "valores" a "chaves"
Valores: posição ou índice onde o elemento se encontra.
Chave: parte da informação que contém o elemento a ser manipulado.

Estrutura de Dados (Grafos)

São estruturas que permitem programar a relação entre objetos. Os objetos são nós do grafo. Os relacionamentos são as arestas.
